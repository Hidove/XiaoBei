小北学生体温自动上报系统
===============

> 运行环境要求PHP7.1+，兼容PHP8.0。

监控运行地址：`http://你的域名 + /api/action?token= + 你的token`

## 下载

演示地址：<http://xiaobei.dalaola.com>
github：<https://github.com/Hidove/XiaoBei>

### token设置

根目录.env文件CONFIG下的token值
